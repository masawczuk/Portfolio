Chest X-Ray Classification — CNN vs MobileNetV2 vs EfficientNet

77–78%Best Accuracy
0.94Best AUC
25,553X-Ray Images

Comparative analysis of three deep learning models for chest X-ray classification (Normal, Pneumonia, Tuberculosis). The project prioritises clinical safety by maximising recall and minimising false negatives.

Project Goal

Design and evaluate three deep learning architectures — a custom CNN, MobileNetV2 and EfficientNet. The primary objective was not only high statistical accuracy, but clinical usefulness: reducing the risk of misclassifying pathological cases as healthy (false negatives).

Research Questions
  • Can deep learning effectively classify chest X-ray images?
  • Does increasing architectural complexity improve classification performance?
  • Can confusion matrix analysis reveal systematic model errors?
Dataset
  • 25,553 X-ray images split into Train / Validation / Test
  • Classes: Normal, Pneumonia, Tuberculosis
  • Preprocessing: normalisation and contrast standardisation
  • Balanced validation and test splits
Model Comparison
  • Custom CNN: 74.39% accuracy · AUC 0.928
  • MobileNetV2: 76.33% accuracy · AUC 0.927
  • EfficientNet: 77–78% accuracy · AUC 0.94
  • Full evaluation: Precision, Recall, F1-score, AUC
  • Confusion matrices & classification reports analysed
Key Insights
  • EfficientNet achieved the highest overall performance
  • Prioritising recall improved clinical safety
  • Normalisation reduced prediction variance across images
  • Confusion matrix analysis revealed typical misclassification patterns
